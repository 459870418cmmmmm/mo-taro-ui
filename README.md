# README

## Development

### Install Dependencies

```
pnpm install
```

### Locally Test

From root:

```
pnpm link --global --dir=./
```

From target project

```
pnpm link --global mo-taro-ui
```

### Lint

```
npm run lint
```

### Unit Test

```
npm run test
```

### Build

```
npm run build
```

### Generate Changelog

```
npm run changelog
```

### Docs

```
npm run start
```
