# Drawer

| 属性 | 是否必填 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- | --- |
| className | 否 | - | `string` | - |
| isOpen | 否 | 是否打开 | `boolean` | - |
| width | 否 | 宽度 | `string` | `150px` |
| position | 否 | 位置 | `left` `right` | `left` |
| onClose | 否 | 关闭回调 | `()=>void` | - |
| children | 否 | - | `React.ReactNode` | - |
