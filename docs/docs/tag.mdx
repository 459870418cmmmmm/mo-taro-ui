# Tag

| 属性 | 是否必填 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- | --- |
| className | 否 | - | `string` | - |
| type | 否 | 类型 | `solid` `outline` | `solid` |
| closable | 否 | 是否可关闭 | `boolean` | `true` |
| onClick | 否 | 点击回调 | `()=>void` | - |
| onClose | 否 | 关闭回调 | `()=>void` | - |
| children | 否 | - | `React.ReactNode` | - |
