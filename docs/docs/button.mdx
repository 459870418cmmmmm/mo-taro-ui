# Button

| 属性 | 是否必填 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- | --- |
| className | 否 | - | `string` | - |
| size | 否 | 大小 | `medium` `small` | `medium` |
| type | 否 | 类型 | `primary` `default` `cartier` | `primary` |
| fill | 否 | 填充模式 | `solid` `outline` `none` | `solid` |
| onClick | 否 | 点击回调 | `()=>void` | - |
| children | 否 | - | `React.ReactNode` | - |

其余属性同 Taro Button
* default目前仅支持solid
