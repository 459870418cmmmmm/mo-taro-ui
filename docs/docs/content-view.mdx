# ContentView

| 属性 | 是否必填 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- | --- |
| safeFooter | 否 | 自动判断是否需要footer | `boolean` | `true` |
| withDefaultPaddingTop | 否 | 是否需要padding top | `number` | `true` |
| extraPaddingTop | 否 | 额外的padding top | `number` | `0` |
| paddingBot | 否 | padding-bottom | `number` | - |
| children | 否 | - | `React.ReactNode` | - |

其余属性同 Taro View
