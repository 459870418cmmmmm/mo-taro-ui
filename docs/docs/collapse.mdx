# Collapse

| 属性 | 是否必填 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- | --- |
| className | 否 | - | `string` | - |
| accordion | 否 | 手风琴 | `boolean` | `false` |
| activeKey | 否 | 选择的key | `string` | - |
| defaultActiveKey | 否 | 默认选择的key | `string` | - |
| icon | 否 | 自定义icon | `React.ReactNode` | - |
| onChange | 否 | onChange回调 | `(val)=>void` | - |
| children | 否 | - | `React.ReactNode` | - |
