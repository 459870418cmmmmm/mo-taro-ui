# Indicator

| 属性 | 是否必填 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- | --- |
| className | 否 | - | `string` | - |
| total | 是 | 总数 | `number` | - |
| currentIndex | 是 | 当前index | `number` | - |
| type | 否 | 类型 | `dot` `line` `groove` `grid` | `dot` |
| color | 否 | 颜色 | `black` `white` `red` | `black` |
