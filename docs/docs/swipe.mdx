# Swipe

| 属性 | 是否必填 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- | --- |
| className | 否 | - | `string` | - |
| name | 否 | 名字 | `string` `number` | - |
| leftAction | 否 | 左边 | `ReactNode` | - |
| rightAction | 否 | 右边 | `ReactNode` | - |
| disabled | 否 | 是否禁用 | `boolean` | `false` |
| onOpen | 否 | 打开时回调 | `(name, position)=>void` | - |
| onClose | 否 | 关闭时回调 | `(name, position)=>void` | - |
| onActionClick | 否 | 点击action回调 | `(e, position)=>void` | - |
| onTouchStart | 否 | 触摸开始回调 | `(e)=>void` | - |
| onTouchEnd | 否 | 触摸结束回调 | `(e)=>void` | - |
| onTouchMove | 否 | 触摸时回调 | `(e)=>void` | - |
| children | 否 | children | `ReactNode` | - |
