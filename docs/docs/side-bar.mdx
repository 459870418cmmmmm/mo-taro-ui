# SideBar

| 属性 | 是否必填 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- | --- |
| className | 否 | - | `string` | - |
| activeKey | 否 | 选择的key | `string` | - |
| defaultActiveKey | 否 | 默认选择的key | `string` | - |
| onChange | 否 | onChange回调 | `(val)=>void` | - |
| children | 否 | - | `React.ReactNode` | - |

## SideBar.Item

| 属性 | 是否必填 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- | --- |
| className | 否 | - | `string` | - |
| title | 是 | 标题 | `string` | - |
| key | 是 | - | `number` `string` | - |
| children | 否 | - | `React.ReactNode` | - |
