# Radio

| 属性 | 是否必填 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- | --- |
| className | 否 | - | `string` | - |
| checked | 否 | 选中状态 | `boolean` | - |
| defaultChecked | 否 | 默认选中状态 | `boolean` | - |
| value | 否 | Group使用时的值 | `string` `number`  | - |
| onChange | 否 | onChange回调 | `(val)=>void` | - |
| children | 否 | - | `React.ReactNode` | - |

## Radio.Group

| 属性 | 是否必填 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- | --- |
| value | 否 | 选中的值 | `string` `number` | - |
| defaultValue | 否 | 默认选中的值 | `string` `number` | - |
| onChange | 否 | onChange回调 | `(val)=>void` | - |
| children | 否 | - | `React.ReactNode` | - |
