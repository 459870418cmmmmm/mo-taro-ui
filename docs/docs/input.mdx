# Input

| 属性 | 是否必填 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- | --- |
| value | 否 | 输入值 | `string` | - |
| defaultValue | 否 | 默认输入值 | `string` | - |
| clearable | 否 | 是否有清空按钮 | `boolean` | `false` |
| min | 否 | type=number时最小值 | `number` | - |
| max | 否 | type=number时最大值 | `number` | - |
| onChange | 否 | onChange回调 | `()=>void` | - |
| onClear | 否 | onClear回调 | `()=>void` | - |
| children | 否 | - | `React.ReactNode` | - |

其余属性同 Taro Input
